# Rustymail

Rustymail is a modern, fast, and secure desktop email client built heavily on Rust and Tauri. It offers a sleek user interface, deep integration with Gmail via direct API synchronization, and offline capabilities. The app provides a fluid experience with lightweight resource usage, using the native speed of rust to give you Gmail usability in a light package.

## Features
- Deep Gmail synchronization with offline support
- Seamless Google account authentication
- Infinite scroll for effortless email navigation
- Light and Dark mode automatic themes
- Highly customizable app settings
- Fully native window management using Tauri

## Stack
- Tauri 2.x
- Svelte 5 / SvelteKit
- Rust
- SQLite (sqlx) for local data caching
